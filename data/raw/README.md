# Raw Data

Expected mentor-aligned inputs:

- `dfsar/l_band/`: Chandrayaan-2 DFSAR L-band full-pol products from PRADAN
- `dfsar/s_band/`: Chandrayaan-2 DFSAR S-band full-pol products from PRADAN
- `dfsar/midas_outputs/`: MIDAS outputs such as `cpr.bin`, `dop.bin`, and any retained C2 products
- `ohrc/`: Chandrayaan-2 OHRC imagery for crater morphology and landing-site assessment
- `shadowcam/`: NASA ShadowCam imagery for doubly shadowed crater analysis
- `dem/`: LOLA or equivalent DEM raster, ideally 5 m/pixel where available
- `illumination/`: PSR masks, shadow products, or solar-geometry layers

Populate this directory before Phase 2 onward.
